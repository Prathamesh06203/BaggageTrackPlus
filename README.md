## ğŸ“Š API Endpoints

### **Sensor Data**
- **POST** `/api/sensor-data`  
  Add sensor readings like acceleration and temperature.  

- **GET** `/api/sensor-data/history`  
  Retrieve historical sensor data for a specific device.

### **GPS Data**
- **POST** `/api/gps-data`  
  Add GPS data like latitude and longitude.  

- **GET** `/api/gps-data/history`  
  Retrieve historical GPS data for a specific device.
